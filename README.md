## Quick Setup

1. Install dependencies:
   ```bash
   npm install
   cd frontend && npm install
   ```

2. Copy environment file:
   ```bash
   cp .env.example .env
   # Edit .env with your values
   ```

3. Deploy contracts:
   ```bash
   npm run deploy:local
   ```

4. Start frontend:
   ```bash
   cd frontend && npm start
   ```
